Penggunaan Awal
Pada saat adaptor <=12V disambungkan ke sumber listrik, alat akan otomatis menyala. Terdapat teks *debug* pada saat pertama kali alat menyala. *Debug* berisi informasi status SD Card, WiFi, dan NTP. Jika tidak ada masalah pada alat, maka user akan diminta memilih produk yang akan diproses menggunakan tombol yang tersedia, kemudian proses penghitungan produk jadi akan dimulai.

Tampilan

Penggunaan Tombol
Terdapat tiga tombol, no 1, 2 dan 3
Tombol 1 = Select
Tombol 2 = Down
Tombol 3 = Up

Navigasi pemlihan produk menggunakan tombol yang tersedia.
Untuk mereset* alat tekatn 3 tombol secara berbarengan 2 detik, sampai keterangan proses muncul di lcd.

reset = mengirim nilai final penghitungan pada produk jadi ke database, kemudian mereset semua nilai yang tersedia di alat.
